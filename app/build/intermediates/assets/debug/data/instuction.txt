1200.0 -- Масса
3000.0 -- Масса в повороте
2.5 ------ Множитель тяжести
0.0 ------ Центр массы по Х
0.1 ------ Центр массы по У
0.0 ------ Центр массы по Z
70 ------ Погружаемость под воду
0.70 ---- Множитель сцепления
0.90 ---- Потеря сцепления
0.48 ---- Смещение сцепления
5 -------- Количество передач
150.0 --- Максимальная скорость
18.0 ---- Ускорение (мощность)
20.0 ---- Инерция двигателя
F ------- Ведущий мост (см. внизу)
Р ------- Тип двигателя (см. внизу)
4.0 ----- Эффективность торможения
0.80 ---- Распределение тормоза
0 -------- Наличие ABS  (см. внизу)
30.0 ---- Уровень поворота колес
0.8 ----- Жесткость пружин
0.08 ---- Жесткость амортизаторов
0.0 ----- Прыгучесть на скорости
0.45 ---- Верхний предел
-0.25 --- Нижний предел
0.45 ---- Смещение наклона
0.3 ----- Множитель анти-наклона
0.2 ----- Сдвиг места посадки
0.60 ---- Множитель повреждений
10000 -- Стоимость автомобиля
20 ------ Флаги модели
0 ------- Флаги handling'a
1 ------- Вид фар  (см. внизу)
1 ------- Вид фонарей  (см. внизу)
0 ------- Анимационная группа


*- F - Передний привод *- Р - Бензиновый двигатель
   R - Задний привод         D - Дизельный двигатель
   4 - Полный привод         E - Электро-двигатель
 *ABS (0 - нету; 1 - есть)
 *Вид фар: 0 - горизонтальный *Вид фонарей: 0 - горизонтальный
                    1 - маленький                                     1 - маленький
                    2 - большой                                        2 - большой
                    3 - вертикальный                               3 - вертикальный
